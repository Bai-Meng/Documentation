# MySQL SQL语句审核平台

安装使用Yearning请参考[官方文档](https://guide.yearning.io/)



## 添加用户

角色可以选择`提交人`和`操作人`，提交人只能提交SQL申请，而操作人即可提交申请也可审批执行。
